[ENGLISH]
-> INSTALLATION <-
To install the skin simply extract the contents to your root client Teamspeak 3 directory (default for the 32-bit version is "C:\Program Files (x86)\TeamSpeak 3 Client\").
From there on all you have to do is to select the Hivecom skin and icon pack from the design menu. To get there simply navigate to the options menu and select the design
submenu from the list on the left hand side. You will also need to install the custom font if you want it to show up inside of Teamspeak. To do so, right click the font
contained within this archive (Aldrich-Regular.tff) and select the install option from the context menu.

[DEUTSCH]
-> INSTALLATION <-
Um den Skin in Teamspeak 3 zu verwenden müssen vorerst alle Ordner und ihr Inhalt in den installations
Ordner (standardmäßig für die 32-bit Version ist "C:\Program Files (x86)\TeamSpeak 3 Client\") von Teamspeak extrahiert werden. Von dort aus kann man den Skin im Design
raster auswählen. Dieses findet man im Einstellungs-Fenster von Teamspeak welches über die Leiste am oberen Bildschirmrand geöffnet werden kann. Außerdem kann noch der
für den skin angepassten Font installieren. Die Font Datei ist im Hauptverzeichnis dieses Archives. Mit einem Rechtsclick auf dieser lässt sich daraufhin das Kontextmenü öffnen,
wo eine Option zur Installation gefunden werden kann.
